All the files needed for vPOS are included in this folder.
To install and configure vPOS, ensure that you have PHP5, MySQL, and a 
webserver correctly configued and running.

installAssistant contains the java source code necessary to create the
vPOS installation assistant. It is a package so it must be compiled
as a group in the directory above installAssistant.

generateDatabase.php initializes the MySQL database for vPOS. It is recommended
that it is deleted once it is no longer needed.
